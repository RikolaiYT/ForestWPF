Приложение реализовано на WPF (.NET) с использованием паттерна MVVM.  
Поддерживает загрузку данных из текстовых файлов, добавление, удаление, сортировку, поиск деревьев, а также расчёт характеристик лесного участка (количество, плотность, средний возраст и др.).
